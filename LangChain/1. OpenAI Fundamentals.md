# Install OpenAI and Import API Key
---

```shell
python3 -m venv /opt/venv
soirce /opt/venv/bin/activate

pip3 install python-dotenv
```

In Windows, run `.\venv\Scripts\activate` to activate the virtual environment.
Once installed, follow below steps to import the API Key:

1. Create a `.env` file and store the API Key:
```text
OPENAI_API_KEY=enter_api_key_here
```

2. Create a `.gitignore` file to ensure you to not upload the `.env` file to the repository
3. Install the `dotenv` library, `pip3 install python-dotenv`. We will use this to load the API key from `.env` file.

# Hello, World!
---

```python
import os
from dotenv import load_dotenv
from openai import OpenAI

# Loads the API key from .env file
load_dotenv()
api_key = os.getenv("OPENAI_API_KEY")
if not api_key:
	raise ValueError("OpenAI Key not found")

client = OpenAI(api_key=api_key)

prompt = "You are an experienced Python programmer. Expplain how decorators work."
def prompt_engine(prompt):
	response = client.chat.completions.create(
		model = "gpt-4o-mini",
		# messages is a list of dictionaries
		messages = [{"role": "user", content=prompt}],
		# will generate a maximum of 100 tokens
		max_tokens = 100,
		# temp is between 0 (least creative) to 1 (most creative)
		temperature = 0.1
	)
	return response.choices[0].message.content

print(prompt_engine(prompt))
```

If you would like to view the individual tokens generated in the response, use the below code:

```python
for i, token in enumerate(response.choices[0],message.content):
	print(f"{token}")
```
