#langchain 

1. Install Python, test with `python -V` on the version installed
2. Create and activate virtual environment

```sh
python -m venv venv
source ./venv/Scripts/activate
```

3. Install OpenAI

```sh
pip install openai
```

4. Create an OpenAI API  Key
5. Export the key as an environment variable, or store in `.env` file. Below is an example of how we an export is in Windows.
```shell
set OPENAI_API_KEY=enter_key_here
```
